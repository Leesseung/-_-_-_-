데이터 수집 전략 — 기계 데이터 vs 직접 조사 병행
기계·시스템에서 뽑는 데이터
  • ERP 생산 실적 (품목별 일별 생산량)
  • MES 설비 로그 (가동/정지 시간, 순간정지 횟수)
  • 설비 OEE 데이터 (가용률·성능률·품질률)
  • 납기 이력 / W/Order 실적 데이터
  • 불량 데이터 (품질 시스템 연동)
  • 자재 입출고 이력 (ERP 재고 모듈)
직접 현장 조사가 필요한 데이터
  • 공정별 S/T — 직접 스톱워치 측정 (라인별)
  • 작업자 수준 등급 — 담당 반장·팀장 인터뷰
  • 잉여시간 항목 — 현장 관찰 및 작업일보 수집
  • 기종변경 소요시간 Matrix — 실제 변경 관찰
  • 사무 업무 프로세스 — 담당자 직접 인터뷰 & 학습
  • 현장 운영 관행 — 비정형 지식 (미문서화 규칙)
PM 메모 사무 담당자에게 파일·데이터·업무 형식을 받고, 직접 옆에서 배우면서 하나씩 문서화할 예정. 처음부터 완벽한 데이터를 기대하지 않고, 수집 가능한 것부터 시작해 점진적으로 정밀도를 높이는 Agile 방식으로 진행.

Phase 3 · W3–5 · 과제3

①

S/T (표준시간) — 직접 측정 필수

수집 방법 라인·공정별·모델별 스톱워치 측정 (본인 직접)
기계 보완 FPT/ICT/비전 등 설비 로그로 기계 공정 S/T 추출
핵심 원칙 S/T는 공정별이 아닌 공정 × 모델 조합으로 측정해야 정확한 계획 가능
수집 항목방법주기비고
수삽 S/T (모델별)스톱워치 직접 측정 (n=20 이상/모델)전체 도입 후 신모델 시모델별 편차 가장 큼
리터치 S/T (모델별)실적 발생 건 측정 + 비율 산출월별불량률 연동 보정
비전검사 S/T기계 로그 추출프로그램 변경 시모델별 검사 프로그램 등록
FPT 테스트 시간기계 로그 자동 추출월별포인트 수 기반 예측 가능
ICT 테스트 시간기계 로그 자동 추출월별지그 교체 시간 포함 필수
코팅 S/T (방식별)기계: 로그 / 수작업: 직접 측정초기 1회기계·수작업 분리 등록
숙련도 보정 계수모델×작업자 실적 통계분기숙련도 등급 연계
영상 S/T 분석 AI (검토 중) 작업자 스마트폰으로 공정 영상 촬영 → AI가 동작 단계 자동 인식 → S/T 자동 측정. 현재 아이디어 단계로, 1단계 완료 후 별도 검토 예정.
